# Dotfiles
These are my dotfiles

# Install
Clone this repo
```bash
git clone https://github.com/jafow/dotfiles.git
```

Install vim plugins
```bash
cd ~/dotfiles
git submodule update --init
```

Run the install script (just makes a bunch of symlinks)
```bash
. install.sh
```
